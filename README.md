# Ashy Wineish Farms & Wines

Welcome to the official repository for **Ashy Wineish Farms** and its premium wine brand, **Ashy Wineish**. We provide healthy crops, livestock, and exquisite wines for a healthier society, proudly rooted in Owode-Idiroko, Ogun State, Nigeria.

## 🌟 Features

- **Dual Brand Identity**: Seamlessly integrates the rustic, agricultural charm of Ashy Wineish Farms with the elegant, sophisticated aesthetics of Ashy Wines.
- **Responsive Design**: Fully responsive UI tailored for mobile, tablet, and desktop experiences.
- **Smooth Animations**: Powered by Framer Motion for elegant scroll and hover transitions.
- **Modern Tech Stack**: Built with React, Vite, Tailwind CSS, and shadcn/ui.

## 🚀 Tech Stack

- **Frontend Framework**: [React](https://react.dev/) with [TypeScript](https://www.typescriptlang.org/)
- **Build Tool**: [Vite](https://vitejs.dev/)
- **Routing**: [React Router v6](https://reactrouter.com/) (Future-flagged for v7 readiness)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **UI Components**: [shadcn/ui](https://ui.shadcn.com/)
- **Icons**: [Lucide React](https://lucide.dev/)

## 📂 Project Structure

- `/src/components` - Reusable UI components (Navbar, Footer, Buttons, etc.)
- `/src/pages` - Main application pages (`Index.tsx`, `About.tsx`, `Wines.tsx`, etc.)
- `/src/assets` - Static assets including images, logos, and icons.

## 🛠️ Local Development

Follow these steps to run the project locally:

1. **Clone the repository**

   ```bash
   git clone <repository-url>
   cd AstyWineishFarms
   ```

2. **Install dependencies**

   ```bash
   npm install
   ```

3. **Start the development server**
   ```bash
   npm run dev
   ```
